# STLAF Internal IT Ticketing System

A full-stack Internal IT Helpdesk and Asset Management System built for STLAF.

## Tech Stack

### Backend
- ASP.NET Core 10 Web API
- Entity Framework Core
- PostgreSQL (Supabase)
- AutoMapper
- JWT Authentication
- BCrypt Password Hashing
- Swagger / OpenAPI

### Frontend
- React
- TypeScript
- Vite
- Material UI (MUI)
- React Router DOM
- Axios
- TanStack React Query
